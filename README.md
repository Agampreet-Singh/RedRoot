## 📦 Installation

### Requirements

- Python 3.8+
- Unix/Linux or Windows with terminal
- `pip` (Python package manager)
- Admin/root access for some modules

### Install

```bash
git clone https://github.com/yourusername/RedRoot.git
cd RedRoot
python3 installer.py
```
